maku - drone charging port and base station

The high-level goals of the maku project are as follows:

  1. to enable korok drones to autonomously charge while "off-the-grid".
  2. to set up an ad hoc, wireless communications network that interfaces between drones and other maku units.
  3. to allow for accurate drone positioning using the previously mentioned network.
  4. to enable bulk data transfer between drones and maku units.
  5. to provide an interface between co-communicating maku units and the World Wide Web when possible and desired by end user.
